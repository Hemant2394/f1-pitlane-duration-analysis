# 🏎️ F1 Pit Lane Duration Distribution Analysis

## Overview
This project analyzes Formula 1 pit lane durations across multiple circuits and compares how track characteristics influence total pit lane time.

## Visualization

![Formula 1 Total Pit Lane Time Comparison](pj%202.png)
## Circuits Analyzed

* Monaco
* Monza
* Miami
* Las Vegas
* Abu Dhabi

* Matplotlib
